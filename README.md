# Technology stack:
- android sdk
- room database
- flow
- kotlin coroutines
- etc.

# Architecture:
- Clean Architecture
- MVVM/MVI

# Third party API's
- Strava
- Firebase (probably)

# Screens:
- Pace Calculator
- Splits
- Pace Prediction
- Strava OAuth
- Metronome Beats (for training cadence)
